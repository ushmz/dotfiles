# dotfiles

To execute all setup, run following.

```sh
make up
```

Ready for setup scripts.

```sh
make init
```

Put symbolic link of config files.

```sh
make link
```

Configure with `defaults` command (for Darwin).

```sh
make defaults
```

Install all tools with homebrew.

```sh
make install
```

Remove all files and symbolic links.

```sh
make unlink
```
